# Site-Cristais-de-V-nus
Cristais de Vênus é um catálogo fictício de cristais desenvolvido para treinar HTML e CSS. O site apresenta uma navegação básica, cards de produtos, seção “Sobre Nós” e rodapé com informações de contato. Projeto criado com foco em prática de layout, responsividade simples e organização de conteúdo.
